The Geography of Conflict

This project uses data to showcase compelling images of protests, historical wars, and pivotal moments around the world.

Each image comes with the underlying codes in a Pluto Notebooks using Julia. For a detailed guide on how to install and use Pluto.jl you can follow this video.

To access the notebook simply click on the image.

Protest in West Asia (#6)

Notes: this map displays the intensity of protests in West Asia from 1997-2024. The intensity of protests is measured as the protests events in the grid point divided by the total of events (counting those that are not protests) for that country. Each white star represents the capital city.
